# TasksApp
Simple app that allows users to manage tasks.

<h2> Technologies </h2>
<ul>
  <li>Angular</li>
  <li>Firebase</li>
  <li>Express</li>
  <li>Node</li>
  </ul>
<br>

## Requirements

Node
Previous installation of Node is required. If you don't have it, go to <a>https://nodejs.org/es/download/</a>.
The user must have  Node >= 20.0.0 y npm >= 10.9.0 installed.
